## Customer Sandbox

This release lets a customer submit an example AI dependency and receive:

- classification
- decision outcome
- reason
- formal receipt
- revalidation triggers
- buyer rules

Run locally:

```bash
pip install -r requirements.txt
python app/sandbox_app.py
```

Then open:

```text
http://127.0.0.1:5000/
```

## Boundary Statement

This release exposes the **customer-facing proof surface only**.

It does not expose:

- protected Elyria Systems substrate
- protected kernel internals
- private research corpus
- unpublished mathematical derivations
- internal enforcement architecture
- customer confidential data
- production secrets or credentials

The customer receives assurance.  
The board receives a decision object.  
The protected substrate remains sealed.
